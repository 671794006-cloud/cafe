# Coffee Shop POS

This is a Point of Sale (POS) system for a Coffee Shop, built with Vanilla HTML, CSS, and JavaScript.

## Features
- POS Terminal with Product Categories
- Shopping Cart & Tax Calculation
- Customer Loyalty Points System
- Dashboard & Sales Overview
- Premium Dark Coffee Theme (Glassmorphism)

## Deployment
This project is designed to be fully static and can be deployed instantly to services like **Vercel** or **GitHub Pages** without any build step.

### Deploying to Vercel
1. Push this repository to GitHub.
2. Go to Vercel and import the repository.
3. Deploy! (No build commands needed).

## Next Steps (Database)
To implement a real database instead of mock data:
1. Set up a Supabase project.
2. Add your `SUPABASE_URL` and `SUPABASE_KEY` to the JS configuration.
3. Use the generated `supabase.sql` to initialize the database tables.
